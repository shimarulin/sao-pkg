# sao-pkg

> Package SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao pkg my-project
```

### From git

```bash
sao vyacheslav shimarulin/sao-pkg my-project
```
